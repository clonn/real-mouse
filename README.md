#REAL-Mouse

node.js demo project, it can tracking your mouse and show other people when user login.
you can show this demo with your friend.

#requirement

 * Node.js > 0.10.x
 * express > 3.x.x
 * socket.io > 0.9.x

#installation

    git clone https://github.com/clonn/real-mouse.git
    cd real-mouse
    npm i
    node app.js
